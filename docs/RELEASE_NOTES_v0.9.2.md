# AniTechou v0.9.2

本次更新聚焦于“可移植备份”能力补齐，以及若干影响导入与注册体验的问题修复。

## 本次更新

### 数据导入导出（可移植备份）
- 新增可移植备份：导出 `*.zip`（包含 `data.json` + 封面 + 笔记插图）
- 导入 `*.zip` 时自动解包资源并重写路径，跨机器/干净环境也能正常显示封面与笔记插图
- 保留旧版 `*.json` 导入导出兼容

### AI 导入与标签
- 修复 AI 导入时作者标签可能出现重复的问题
- 数据库层新增作品标签唯一约束，并在迁移阶段自动清理历史重复标签，避免再次产生重复

### 注册窗口
- 修复“注册新账号”窗口高度不足导致按钮被遮挡的问题
- 窗口支持调整大小，并在内容超出时提供滚动

## 安装与运行
- 适用系统：Windows 10/11 x64
- 下载并运行安装包：`AniTechou-setup-win-x64.exe`
- 本版本为自包含安装包，无需额外安装 .NET 运行时

## 数据说明
- 软件数据默认保存在 `%LocalAppData%\AniTechou`
- 更新安装包通常不会清除现有账号、数据库、配置与图片资源
