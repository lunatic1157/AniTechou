# AniTechou v0.9.5 Release Notes

v0.9.5 是一次“整理与陪伴”更新：前半段收口标签、网络代理与 AI 推荐质量，后半段补上笔记编辑体验和夜子桌宠 MVP。

## 笔记体验

- Markdown 笔记编辑器从上下分栏改为左右分栏：左侧编辑，右侧实时预览。
- 分栏宽度支持拖拽调整，并会记住上次使用比例。
- Markdown 预览刷新增加短防抖，输入长笔记时不再每个字符都立刻完整重绘。
- Markdig 解析管线改为复用，减少重复初始化开销。
- 保留现有 Markdown 存储、自动保存、图片/链接插入、标签和作品关联逻辑。

## 夜子桌宠 MVP

- 设置页新增“启用夜子桌宠”开关，默认关闭。
- 启用后显示一个透明置顶的夜子桌宠小窗，可拖拽移动，位置会保存。
- 桌宠会响应部分应用事件：
  - AI 思考时进入 `thinking` 状态。
  - 笔记保存成功时进入 `pleased` 状态。
  - AI 或保存失败时进入 `annoyed` 状态。
  - 拖拽时进入 `moving` 状态。
- 桌宠使用 6 个独立透明 PNG 单帧状态，不再使用 WPF 矢量占位图作为主要形象。
- 资源仍放在 `Assets/Pets/Dalian` 目录下以兼容当前加载链路，manifest 角色标识已切换为 `yoruko`。
- 桌宠不再做倾斜或上下晃动，避免静态角色图在桌面上出现违和动画。
- 双击桌宠可打开轻量对话框；对话框支持本地短回复、状态调试、空输入提示、隐藏和关闭桌宠。
- 右键菜单支持打招呼、打开对话框、暂时隐藏和关闭桌宠。

## AI 与资料整理

- 标签清理策略集中到 `TagPolicy`，自动标签会过滤年份、日期、评分、状态等冗余信息。
- 新增全库标签清理预览窗口，用户可先查看变更再应用。
- 网络访问统一走 `NetworkClientFactory`，支持系统代理、自定义代理和无代理模式。
- AI 推荐结果新增推荐分类与推荐理由，推荐会参考本地收藏画像。
- AI 推荐卡片展示推荐分类和理由，减少“只给标题”的空泛结果。
