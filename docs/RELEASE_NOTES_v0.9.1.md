# AniTechou v0.9.1

本次更新主要围绕使用体验、笔记编辑和版本维护做了一轮修正与优化。

## 本次更新

### 作品详情页
- 修复作品详情页中“关联笔记”列表未正常显示的问题
- 优化进度输入体验：当作品存在总集数/总卷数时，只输入数字也会自动补全为 `当前进度/总数`

### 笔记编辑器
- 插图支持拖拽插入与粘贴插入
- 插图支持拖拽移动位置
- 插图支持通过拖动图片边缘/角落调整显示尺寸
- 优化自动保存逻辑，减少输入过程中偶发卡顿
- 自动保存不再频繁触发外层视图刷新

### 设置与版本信息
- 设置页版本号改为自动读取项目版本
- 本次版本同步更新为 `0.9.1`

## 安装与运行
- 适用系统：Windows 10/11 x64
- 下载并运行安装包：`AniTechou-setup-win-x64.exe`
- 本版本为自包含安装包，无需额外安装 .NET 运行时

## 数据说明
- 软件数据默认保存在 `%LocalAppData%\AniTechou`
- 更新安装包通常不会清除现有账号、数据库、配置与图片资源
