# AniTechou v0.9.0

AniTechou 是一款「AI + 笔记驱动」的 ACGN 作品管理工具：不仅记录“我看了什么”，更强调“我为什么喜欢、我当时想到什么”，并通过 AI 帮助搜索、补全和整理作品信息与笔记。

## 核心亮点

### AI：自然语言搜索、推荐、补全与整理
- 支持自然语言搜索和推荐作品
- 支持 AI 批量搜索与导入作品信息
- 支持 AI 完善作品信息，如简介、季度、制作公司/作者、封面等
- 支持对话式意图识别，可辅助处理作品与笔记相关操作
- 支持多平台配置：DeepSeek、智谱AI、阿里云百炼、OpenAI

### 笔记：记录想法，而不只是记录看过
- 富文本笔记编辑
- 自动保存、撤销/重做
- 支持插图、链接、标签、关联作品
- 标签建议与去重
- 兼容旧纯文本读取，并使用更适合富文本的保存形式

## 安装与运行
- 适用系统：Windows 10/11 x64
- 下载并运行安装包：`AniTechou-setup-win-x64.exe`
- 本版本为自包含安装包，无需额外安装 .NET 运行时

## AI 功能说明
- AI 功能需要在“设置”中自行配置 API Key、接口地址与模型
- API Key 仅保存在本机配置文件中，不随安装包分发

## 数据存储位置
- 软件数据默认保存在 `%LocalAppData%\AniTechou`
- 包含配置、账号信息、数据库、封面、头像与笔记插图
- 重新安装通常不会自动清除这些数据

## 已知说明
- 导出数据中若包含本机图片路径，跨机器导入后图片可能需要重新补充
- 首次使用建议优先检查 AI 配置、主题切换与笔记编辑功能是否符合预期
